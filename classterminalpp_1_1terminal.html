Detailed Description

A class that encapsulates a terminal.

The class, together with the set of manipulators provide a way of writing to a terminal. It is agnostic to the actual write method used. The writing method is passed as a WriteContinuation, which is called for the required output.

Read from the terminal.

Usage
Stream in bytes, and the resultant tokens parsed from the stream will be sent to the continuation passed in.
using namespace terminalpp::literals;
void read_tokens(terminalpp::tokens);
void raw_write(terminalpp::bytes);
terminal term{raw_write};
term.read(read_tokens) >> "\\x1B[13~"_tb;
// read_tokens was called with a collection of one token, which
// contained the f3 virtual key.

Write to the terminal.

Usage
Stream in text, or use manipulators to modify the state of the terminal.
void raw_write(terminalpp::bytes);
terminal term{raw_write};
term.write(raw_write) << "Hello, world!"
<< move_cursor({17, 29});
Writing your own manipulators.
A manipulator is streamable to a writer if is has a member function with the following signature:
template <class WriteContinuation>
void operator()(terminalpp::terminal_state &, WriteContinuation &&);
