Detailed Description

Constants for Single Shift to G3 commands.

Note
SS3 is not a command set per se. Instead, it is a shift to another character set for exactly one character. However, since some of the characters in that set perform actions (e.g. cursor movement), it is frequently used as shortcut sequences for those commands.
